Dat0r\Runtime\Document\DocumentList

class DocumentList extends TypedList implements IDocumentChangedListener

DocumentList is a TypedList implementation, that holds IDocuments and provides some extra convenience.

You can attach to it as an IDocumentChangedListener and will be notified on all events occuring from it's contained documents.
